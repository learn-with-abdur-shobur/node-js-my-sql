## Start Node js and MySql

```bash
npm init --y
npm i express mysql dotenv
```

- Create file `app.js`
- connect and test `http://localhost:3030/api`

  **Folder structure**

  ```text
  -index.js
    |-config[]
    |--|-database.js // have database info
    |
    |-api[]
    |--|-user[]
    |--|--|-user.service.js
    |--|--|-user.controller.js
    |--|--|-user.route.js
  ```
